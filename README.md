# BMI Calculator 

An iOS app that calculates Body Mass Index (BMI) based on user height and weight. The app also provides customized health advice depending on the resultâ€”whether the user is underweight, overweight, or within a healthy range.

## Features

- Clean and interactive multi-screen user interface with smooth navigation.
- Real-time BMI calculation based on user input.
- Personalized health suggestions.
- Color-coded feedback to visualize BMI category.

- MVC Architecture
